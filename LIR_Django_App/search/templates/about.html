About CaseSearchIreland


CaseSearchIreland (CSI) is a master's project that explores the use of semantic search within the Irish legal system. By leveraging Natural Language Processing (NLP) techniques, CSI enables users to move beyond traditional keyword-based searches towards legal research that considers the contextual meaning of both the user queries and judgments.

To achieve this, the BERT (Bidirectional Encoder Representations from Transformers) model* was used to generate embeddings for over 17,000 judgments from the Irish Superior Courts (High Court, Supreme Court, and Court of Appeal). These embeddings are used to capture the semantic representations of each judgment in a dense vector space. In addition to this, embeddings are also created for user queries. This enables CSI to consider the contextual meaning of each judgment and query, allowing users to carry out more intuitive searches. To enable effective retrieval, the embeddings created for each judgment are stored in a vector database as this provides fast and efficient comparisons between the embeddings of user queries and judgment embeddings. The judgments returned to the user are done so in ranked order accompanied by a similarity score (%) that quantifies the relevance between the user query and each judgment. Please note that CSI only maintains reported judgments that date to 7th July 2022.

*This project utilised the all-mpnet-base-v2 model, which is a variant the Sentence Transformers model.
